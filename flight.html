    User Research

    A sample of users were used to identify a specific set of needs that are not currently being addressed by popular flight booking meta-search sites.

    User Goal

    The user wants a simple to read interface that allows them to compare prices to a set of personal criteria and weigh the cost to any given flight option, while still maintaining a relationship with specific airlines.
