Flight Booking Concept Website Design

  • Process
    Process
    Research:

    Through a series of interviews, I was able to gather information about what the client goals are when booking a flight through an online website.

    User Gaol: The user wants a simple to read interface that allows them to compare prices to a set of personal criteria and weigh the cost to any given flight option.

    Sketch:

    Through a serious of low-fidelity wireframes, a final wireframe was selected.

    The selected wireframe was digitized in Sketch.

    Prototype:

    An interactive prototype was created in Axure.

    Test:

    A small sample of users were used to test the "Flight Finder" prototype concept.

    A report was created outlinning the inssues users had and the recomcendations for improvement.

    Redesign:

    Based on testing, the recomendations were implemented in Axure.
